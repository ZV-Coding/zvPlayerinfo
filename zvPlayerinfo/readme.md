# zvPlayerinfo

### Features
- See all blips from players on the map
- Displays the player's name above their head
- clear config

#### Installation
1. Drag & Drop the Script to your server
2. Open your Server.cfg / CFG Editor in TxAdmin
3. Add "ensure zvPlayerinfo" to your cfg
4. Enjoy it!
